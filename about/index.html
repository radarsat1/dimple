About Dimple

Dimple is the Dynamically Interactive Musically PhysicaL Environment.

Blank canvas for creating audio-haptic interactive environments.

Physics by Open Dynamics Engine.

Haptics and graphics by Chai3d.

Work with any media software that supports Open Sound Control.

  • Pure Data
  • Max/MSP
  • SuperCollider
  • ChucK

Operating systems:

  • Linux
  • Mac OS X
  • Windows

Haptic devices:

  • Any supported by chai3d including,
    • Phantom line (Geomagic)
    • Delta line (Force Dimension, Novint)
    • Soon: Haply
  • Also input devices such a Leap Motion

Open source.
